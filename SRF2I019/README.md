### Description

Internal 868 & 915 MHz ISM / Sigfox / LoRa / LPWA Passive Self-Adhesive Flexible PCB Antenna.

A self-adhesive FPC antenna for the ISM bands. Connected via an MHF IPEX connector. Available in a 100mm cable length. Other cable lengths are available, but minimum order quantities apply.
