### Description

The TPSM84338 is a high efficiency, high voltage input, and easy-to-use synchronous buck power module with high design flexibility. With the wide operating input voltage range of 3.8V to 28V, the TPSM84338 is designed for systems powered from 5V, 12V, 19V, 24V power-bus rails. The device supports up to 3A continuous output current and 98% maximum duty cycle.

TPSM84338 uses fixed frequency peak current control with internal compensation for fast transient response and good line and load regulation. With the optimized internal loop compensation, the device eliminates the need for external compensation over a wide output voltage range and switching frequency.

TPSM84338 allows high design flexibility with wide supported switching frequency of 200kHz – 2.2MHz set by the RT/SYNC pin. The device has the option for pulse frequency modulation (PFM), forced continuous conduction modulation (FCCM) at light load condition, and adjustable soft-start (SS) time / power-good (PG) indicator with different configuration of the MODE pin.

TPSM84338 provides protection functions including thermal shutdown, input undervoltage lockout, cycle-by-cycle current limit, and hiccup short-circuit protection. TPSM84338 is available in 9-pin, 3.3mm × 4.5mm, QFN package, and the pinout is available for single-layer PCB layout and the junction temperature is specified from –40°C to 150°C.
